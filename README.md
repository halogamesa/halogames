欢迎使用我们开发的游戏999秒!幸存者。我们非常重视用户的隐私保护，本政策旨在向您说明在您使用本游戏时，我们如何处理您的个人信息。

一、我们如何收集和使用信息
为了提供基本的游戏功能和服务，我们仅收集实现业务功能所必需的信息：

游戏运行所必需的信息：为保障游戏正常运行，识别账号异常状态，我们会收集您的设备型号、操作系统版本、唯一设备标识符（如 OAID/IMEI）。

账号与实名服务：本游戏使用 TapTap SDK 提供登录与实名认证服务。

二、第三方 SDK 说明（重要）
为保障游戏功能的实现与状态稳定，我们接入了第三方提供的 SDK。

SDK 名称：TapTap SDK (易玩网络)

使用目的：提供 TapTap 账号登录、实名认证、防沉迷系统。

收集信息类型：设备标识符（OAID、IMSI）、网络信息、实名认证信息（实名信息由 TapTap 直接处理，开发者不触达）。

官网隐私政策链接：https://www.taptap.com/privacy-policy

三、信息存储与安全
我们仅在本政策所述目的所必需的时间内保留您的信息。我们采用行业标准的安全技术来防止您的信息丢失、被盗用或遭未经授权的访问。

四、您的权利与账号注销
您有权访问、更正或删除您的个人信息。如需注销账号，您可以直接在 TapTap 客户端内进行账号注销操作，或通过以下联系方式与我们联系，我们将在 15 个工作日内核实并处理。

五、未成年人保护
我们严格遵守国家关于未成年人防沉迷的法律法规。未成年人用户必须在监护人的指导下阅读本协议，并接入防沉迷系统。
